# Quality Assurance Checklist

## Pre-Design Review Checklist

### ✅ Product Information
- [ ] Product name finalized
- [ ] Product category confirmed
- [ ] All specifications documented
- [ ] Key features listed (minimum 5)
- [ ] Target audience defined
- [ ] Competitor analysis complete

### ✅ Marketing Copy
- [ ] Five selling points written
- [ ] Each point has headline + description
- [ ] All claims verified as truthful
- [ ] Copy is benefit-focused
- [ ] Language is US English
- [ ] No spelling/grammar errors

### ✅ Design Style Guide
- [ ] Brand colors selected (primary, secondary, accent)
- [ ] Typography finalized (headline, body, caption fonts)
- [ ] Photography style defined
- [ ] Tone and mood established
- [ ] Reference images collected (minimum 5)
- [ ] Competitor styling reviewed

## Visual Quality Checklist

### Image 01 - Main Product Image

**Product Quality**
- [ ] Product is sharp and in focus
- [ ] All key features clearly visible
- [ ] Product is positioned optimally
- [ ] Product fills 60-70% of frame
- [ ] Best angle showing key details
- [ ] Product is undamaged/perfect condition

**Background & Lighting**
- [ ] Background is clean and professional
- [ ] Lighting is even and professional-quality
- [ ] Shadows are minimal or natural
- [ ] No harsh glare or blown-out areas
- [ ] Background color is consistent (white or style guide)
- [ ] No distracting elements

**Overall Appearance**
- [ ] Image looks professional and retail-ready
- [ ] Color grading is accurate to product
- [ ] Contrast is appropriate
- [ ] Image is appealing and inviting

### Images 02-06 - Selling Point Images

**Product/Focus Element**
- [ ] Main subject (product or feature) is sharp
- [ ] Subject is properly lit and visible
- [ ] Close-ups show detail clearly
- [ ] Product angle supports the selling point
- [ ] Subject occupies appropriate frame space
- [ ] Visual supports the messaging

**Text Overlay Quality**
- [ ] Text is crisp and sharp
- [ ] No anti-aliasing artifacts
- [ ] Text color has high contrast with background
- [ ] Headline is prominent (larger)
- [ ] Supporting text is readable
- [ ] Text is positioned clearly (not overlapping image)

**Text Accuracy**
- [ ] Headline matches selling point theme
- [ ] Supporting text matches marketing copy
- [ ] No spelling errors
- [ ] No grammar errors
- [ ] Capitalization is consistent
- [ ] Punctuation is correct

**Background & Design**
- [ ] Background complements the message
- [ ] Text overlay background is semi-transparent and readable
- [ ] Overall design is clean and professional
- [ ] Design supports brand aesthetic
- [ ] Lighting is consistent with image set

### Images 07-08 - Lifestyle Scene Images

**Scene Composition**
- [ ] Scene shows realistic use case
- [ ] Product is naturally positioned
- [ ] Scene is visually interesting
- [ ] Setting is appropriate to use case
- [ ] Multiple elements create engaging scene
- [ ] Composition guides eye to product

**Lighting & Color**
- [ ] Lighting is professional and natural
- [ ] Color temperature is warm and inviting
- [ ] Lighting is consistent across both images
- [ ] Shadows and highlights are balanced
- [ ] Color grading is rich but not oversaturated

**Product Visibility**
- [ ] Product is clearly visible in scene
- [ ] Product is not obscured or hidden
- [ ] Product is in focus or appropriately blurred
- [ ] Product context is clear (what's it used for?)

**Scene Styling**
- [ ] Props and accessories look natural
- [ ] Scene feels staged but realistic
- [ ] Styling matches brand aesthetic
- [ ] Mood and emotion are clearly conveyed
- [ ] Scene is aspirational and appealing
