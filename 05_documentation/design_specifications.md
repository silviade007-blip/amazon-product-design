# Complete Design Specifications

## Overview
This document contains the complete technical and creative specifications for all 9 Amazon product images.

---

## Image Specifications (All Images)

### Technical Requirements
- **Dimensions**: 1600 x 1600 pixels (must be exact)
- **Aspect Ratio**: 1:1 square
- **Resolution**: 72 DPI (for web/digital)
- **Color Space**: RGB (not CMYK)
- **File Format**: PNG (preferred for transparency) or JPEG (for compatibility)
- **File Size**: Maximum 2MB per image
- **Color Profile**: sRGB

### Quality Standards
- **Sharpness**: Images must be sharp and clear at 1600px
- **Compression**: Minimal compression, no visible artifacts
- **Anti-aliasing**: Apply to all text and edges
- **Color Accuracy**: True to product and brand colors

## Text Overlay Standards

### Typography for Overlays

**Headline (Primary Text)**
- Font Size: 48-60pt (at 1600px canvas)
- Font Weight: Bold or Black
- Font Family: Sans-serif (Montserrat, Poppins, Gotham)
- Letter Spacing: Normal or slightly expanded
- Color: High contrast (white or dark)
- Line Height: 1.2

**Supporting Text (Secondary)**
- Font Size: 24-32pt (at 1600px canvas)
- Font Weight: Regular or Medium
- Font Family: Same as headline or complementary
- Letter Spacing: Normal
- Color: High contrast, slightly less emphasis
- Line Height: 1.3

**Captions (Tertiary)**
- Font Size: 16-20pt
- Font Weight: Light or Regular
- Font Family: Same as supporting
- Color: Readable but less prominent

### Text Placement Rules

✅ DO:
- Place text in corners or edges
- Maintain 40px margin from image border
- Use semi-transparent background for readability
- Ensure legibility at thumbnail size (500px)
- Balance text weight in composition

❌ DON'T:
- Cover important product details
- Use small text on busy backgrounds
- Place text in exact center
- Use excessive punctuation
- Use multiple colors for single message
